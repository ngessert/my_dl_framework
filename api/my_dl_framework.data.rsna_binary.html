CV split generation script for rsna challenge. Usage:

define_cv_split_rsnachallenge.py [-h] –num_cv=<num_cv> –save_path=<save_path>

Options:

-h –help Display help –num_cv=<num_cv> Number of cv splits –save_path=<save_path> file.json containing the defined splits
