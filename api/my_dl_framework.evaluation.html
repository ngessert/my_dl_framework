my_dl_framework.evaluation.eval_and_predict module¶

Prediction script. Works with or without targets. If there are targets, get metrics as well. Options:

--folder=<folder>

folder to run prediction on

-cl –clearml use clearml.

Example

python my_dl_frameworkevaluationeval_and_predict.py –folder=C:dataRSNA_challengeexperimentstest_config.yaml20-02-2022-20-38-12 -cl clearml
